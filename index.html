    This is a PALINDROME Game.
    Each player has to click PROMPT BOX button turn by turn,then a prompt will appear you have to type a word which should be a palindrome if word is palindrome then player who gave the correct answer will get a point and if player dosent give the correct answer then thier will be no deduction of point. The player who scores 10 first will be the winner.



    PALINDROME IS A WORD WHICH WHEN REVERSED REMAINS THE SAME EXAMPLE MADAM IS PALINDROM, BUT HELLO ISN'T AS HELLO WILL RETURN OLLEH
